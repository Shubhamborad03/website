# Wolf Partners — Roofer Demo

A scroll-driven sales demo showcasing Wolf Partners' AI automation stack for QLD residential roofing contractors.

## Stack
- Next.js 16 (App Router) + React 19 + TypeScript
- Tailwind CSS v4
- Framer Motion 11 (scroll + reveal animations)
- Lenis (smooth scroll)
- Lucide React (icons)

## Sections
1. **Hero** — `12 leads → 47 leads. Same crew.` transformation framing
2. **Quote engine** — AI roof scan with viewfinder, edge measurements, damage callouts
3. **Conversation** — AI replying to leads in the operator's voice
4. **Pipeline / CRM** — Live dashboard with KPI cards, bar chart, recent leads, and a territory coverage map (real satellite imagery + 82 house markers across 6 CRM statuses)
5. **Social listening** — Brand24-style feed catching "anyone got a roofer" posts
6. **Content engine** — One job becomes 24 pieces (carousels, reels, stories, ads, etc.)
7. **The math** — Animated +$66K hero number + dark/cream vertical comparison table
8. **Contact** — Wolf Partners logo + dave@wolfpartners.com.au

## Run locally
```bash
npm install
npm run dev
```

## Deploy
Built for Vercel. Point Vercel at this folder as the project root, accept defaults — no env vars needed.
